# find_apis_for_entity

Find all API nodes that act on a given entity/concept. Uses `ACTS_ON` edges to return every API that operates on the specified concept.

## Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `entity` | `string` | Yes | Entity canonical name (e.g., `"Workspace"`) or concept node ID |

## Response

Returns an array of full ApiNode objects:

```json
[
  {
    "id": "eb0e9f44ca78",
    "type": "api",
    "operation_id": "create_workspace",
    "method": "POST",
    "path": "/v1/workspaces",
    "resource": "Workspace",
    "capabilities": ["create", "provision"],
    "preconditions": ["user_authenticated", "billing_plan_active"],
    "permissions": ["workspace:write"],
    "side_effects": ["creates_workspace_record", "sends_welcome_email"]
  }
]
```

## Examples

```
// By canonical name
find_apis_for_entity({ entity: "Workspace" })

// By node ID
find_apis_for_entity({ entity: "9c4c9063760e" })
```
